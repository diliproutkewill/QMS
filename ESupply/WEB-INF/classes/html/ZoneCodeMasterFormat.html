Zone Code Upload - File Format
Important Points
1 The Zone Code upload data is entered under 2 headers
2 First header is used to enter MASTER data for the Zone Codes
3 The user can also enter multiple MASTER data & corresponding DETAILS data in the same sheet
4 The user can enter any number of DETAILS data
5 First header is used to enter MASTER data for the Zone Codes
7 For multiple MASTER data, the user can specify a matching ROW.NO under MASTER & DETAILS data as a reference number
e.g In the above example, Zone Codes with Sl.No.1 corresponds to Zip Code Type N while Sl.No.2 corresponds to Zip Code Type AN
8 There should be one empty row between Master & Detail
9 For Zip Code Type Numeric you Should enter N
Alpha Numeric you Should enter AN
10 For Numeric ,Alphanumeric Value Should be '-'
