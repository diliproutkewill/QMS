Location - File Format
1) First Row should be Header.
2) Every column should have a tab space.
3) Location Id and Country Id can be of maximum 3 & 2 characters respectively.
4) Valid numeric values for SHIPMENT MODE are as follows:
  1 - Air
  2 - Sea
  3 - Truck
  4 - Air & Sea
  5 - Air & Truck
  6 - Sea & Truck
  7 - Air, Sea & Truck
5) If Allow Modify is not selected then only new location details will be added provided that the location id does not already exist.
6) If Allow Modify is selected then it will update the previous location details provided that the location id already exists. 
7) If "Export exceptions to excel file" is checked then it will export all the un-uploadable charge details to an excel file with a extra remarks column which contains the reasons for not uploading that record.
8) User can correct the remarks accordingly and upload the generated file, removing the remarks column.
9) Show Report will generate an excel report showing all the data accordingly.

Note : 
 1. Open Microsoft Excel Sheet and enter data as mentioned above. 
2. Save the file in 'Comma Delimited' (.CSV) format .
3.* Denotes Compulsory Fields.
4.Comma's are not allowed within a cell.
5.If there is any empty Row in between the file then it will be treated as End Of File and the records after the empty row will not be considered for upload.
