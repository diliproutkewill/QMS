Port - File Format
1) First Row should be Header.
2) Every column should have a tab space.
3) Port Id, Port Name,Country Id can be of maximum 15, 50, 2 respectively.
4) If Allow Modify is not selected then only new port details can be added.
5) If Allow Modify is selected then it will update the previous port details
6) If "Export exceptions to excel file" is checked then it will export all the un-uploadable country details to an excel file with a extra remarks column which contains the reasons for not uploading that record.
7) User can correct the remarks accordingly and upload the generated file, removing the remarks column.
10) Show Report will generate an excel report showing all the data accordingly.

Note : 
 1. Open Microsoft Excel Sheet and enter data as mentioned above. 
2. Save the file in 'Comma Delimited' (.CSV) format .
3.* Denotes Compulsory Fields.
4.Comma's are not allowed within a cell.
5.If there is any empty Row in between the file then it will be treated as End Of File and the records after the empty row will not be considered for upload.

Example :

PORT_ID*  

PORT_NAME*

COUNTRYID*  

DESCRIPTION
PORT1 PORT ONE IN  
PORT2 PORT TWO IN  
PORT3 PORT THREE IN  
PORT4 PORT FOUR IN  
PORT5 PORT FIVE IN  
PORT6 PORT SIX IN
