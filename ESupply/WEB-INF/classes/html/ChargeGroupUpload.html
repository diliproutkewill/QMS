Charge Grouping Upload - File Format

 

  1. First Row should be Header.
  2. Every column should have a tab space.
  3. The format differs for Process - Add/Modify.
  4. The user can upload Charge Group Ids for Shipment Mode - Air, Sea, and Truck in case of Process - Add.   
  5. Shipment Mode should be separated by ‘/’. For e.g. Air/Sea/Truck.
  6. In case of Modify, Terminal Id for the corresponding Charge Group Id should be provided. If left blank, the Logged-In Terminal Id will be used.
  7. The Maximum length for Charge Group ID is 40 characters.
  8. Any number of Charge Group Ids can be uploaded at a single time. (Please Refer the Example Shown).
  9. If ‘Export exceptions to excel file’ is checked then all the records which could & could not be uploaded will be exported to an Excel File with appropriate remarks shown in an additional column.

Note:

  1. Open Microsoft Excel Sheet and enter data as shown below.
  2. Save the file in 'Comma Delimited' (.CSV) format.
  3. All fields are Mandatory.
  4. Commas are not allowed within a cell.
  5. If there is any empty Row in between the file then it will be treated as End Of File and the records after the empty row will not be considered for upload.
