Canada Zone Code Upload - File Format 
Important Points
  1. Canada Zone Code upload data is entered under 2 headers.
  2. First header is used to enter MASTER data for the Zone Codes.
  3. The user can also enter multiple MASTER data & corresponding DETAILS data in the same sheet .
  4. The user can enter any number of DETAILS data.
  5. First header is used to enter MASTER data for the Zone Codes.
  6. For multiple MASTER data, the user can specify a matching ROW NO under MASTER & DETAILS data as a reference number.
  7. There should be one empty row between Master & Detail Data.
Example
ROW ID Shipment Mode Console Type Location TerminalId City State
1 Air YUL DHLYUL Montreal
2 Sea FCL YOW DHLYOW OTTAWA
