Commodity - File Format
1) First Row should be Header.
2) Every column should have a tab space.
3) Commodity Id, Description,Handling Info,Commodity Type and Hazard Indicator can be of maximum 5, 50,2000,100 and 1 of length respectively.
4) Valid values for Hazard Indicator are: Y and N
5) Valid values for Commodity Type are:
    1     For Edible,Animal & Vegetable Products
    2     For AVI,Inedible Animal & Vegetable Products
    3     For Textiles,Fibres & Mfrs.
    4     For Metals,Mfrs Excl M/C , Vehicles & Electrical Equipment
    5     For Machinery,Vehicles & Electrical Equipment
    6     For Non-Metallic Minerals & Mfrs.
    7     For Chemicals & Related Products
    8     For Paper,Reed,Rubber & Wood Mfrs.
    9     For Scientific,Professional & Precision Instruments
   10    For Miscellaneous
6) If Allow Modify is not selected then only new currency details can be added.
7) If Allow Modify is selected then it will update the previous currency details
8) If "Export exceptions to excel file" is checked then it will export all the un-uploadable country details to an excel file with a extra remarks column which contains the reasons for not uploading that record.
9) User can correct the remarks accordingly and upload the generated file, removing the remarks column.
10) Show Report will generate an excel report showing all the data accordingly.

Note : 
 1. Open Microsoft Excel Sheet and enter data as mentioned above. 
2. Save the file in 'Comma Delimited' (.CSV) format .
3.* Denotes Compulsory Fields.
4.Comma's are not allowed within a cell.
5.If there is any empty Row in between the file then it will be treated as End Of File and the records after the empty row will not be considered for upload.
